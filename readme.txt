An easy to use and light plugin for creating popup with user friendly interface. 

== Description ==
Users can create multiple popups and set time-out. They can also set popup on page visit or leave.There is no limit on creating popups. 

== Installation ==
Installation is fairly straight forward. Install it from the WordPress plugin repository.
